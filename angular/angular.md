# ANGULAR PORTFOLIO

Included in this folder is some sample code and a short video displaying my initial dive into the world of Angular.

I focused on building a simple and intuitive UI for an online food ordering service. I used a loosely coupled component architecture for this project and implemented Angular Forms for many of the menu components.

If you are interested in the full repository, it can be found here: https://github.com/loganchester/online-ordering-app.
