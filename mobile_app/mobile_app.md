# Mobile App

During my first semester of third year, I took an introduction to software engineering class. 
In this class, we were tasked with designing and implementing a mobile application as specified by a product owner. 
I acted as Scrum Master for our team of 7 and, using technologies like React Native, Java, and MongoDB, we implemented a simple travel itinerary application.

Although I was a part of the front-end team and focused on features such as the filtering components and Google Maps API integration, my expertise in setting 
up a Spring application was utilized as well. Much of the backend code was based on previous RESTful services I have developed (I have included sample code
for one of these in the "Coding Samples" folder).

Included in this folder are pictures of the app and React Native/JavaScript coding samples.
